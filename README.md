# Tienda Online de Productos de belleza

Este proyecto es una página web ficticia desarrollada como parte del curso de Front-End.  
La tienda ofrece productos de belleza (maquillajes, perfumes) y está organizada en secciones semánticas: Inicio, Productos, Reseñas y Contacto.  

### Funcionalidades:
- Barra de navegación interna
- Sección de productos con tarjetas responsivas
- Sección de reseñas con CSS Grid
- Formulario de contacto conectado a Formspree
- Footer con iconos de redes sociales

El objetivo es practicar la construcción de un sitio completo y prepararlo para futuras mejoras
